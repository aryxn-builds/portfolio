## вңЁ Features

### рҹӨ– AI Chat Assistant
A real AI chatbot embedded in the portfolio powered by the **Anthropic Claude API**. Visitors can have actual conversations with an AI trained on my personality, projects, and background. Responds in streaming mode with typing animations and maintains full conversation history.

### рҹ“· Live Computer Vision Demo
Real-time face detection and pose estimation running **100% client-side** using **MediaPipe.js**. No data ever leaves your device. Features include 468-point face mesh detection, full-body pose estimation, eye tracking with gaze direction, and live inference metrics at 30fps.

### рҹҺЁ Neural Signature Art Generator
Visitors enter their name and the portfolio generates a **unique deterministic art piece** seeded from their name characters. Every name produces a different neural network visualization. Downloadable as PNG with a pre-filled LinkedIn share button.

### рҹҢі RPG Skill Tree вҖ” Certifications
Certifications displayed as an **interactive RPG skill tree** with 4 tracks (ML Foundation, GenAI, Competitions, Future Path). Unlocked nodes glow orange/blue, locked nodes show the learning roadmap. Animated traveling dots on connection lines and particle burst on hover.

**Certifications included:**

### рҹҗҷ Live GitHub Activity Feed
Real-time GitHub commit feed pulling from the **GitHub REST API** through Next.js serverless proxy routes (bypassing CORS). Features a 30-day contribution graph, live commit messages, repository color coding, and auto-polling every 60 seconds.

### рҹҺҜ Recruiter Mode Toggle
A single toggle (or press **R**) that activates a recruiter-optimized experience: availability badge, one-click resume download, skills summary, and section navigator вҖ” all in a slide-in sidebar panel. Auto-activates when visitors arrive from LinkedIn.

### рҹ–ҘпёҸ Terminal Contact Section
Contact section styled as a live SSH terminal session with character-by-character typing animation. JSON contact data types out on scroll entry. Orbital node diagram with spinning rings, fixed contact nodes, and animated traveling dots connecting to the AY core.

## рҹҡҖ Getting Started

### Prerequisites
```bash
node >= 18.0.0
npm >= 9.0.0
```

### Installation

```bash
# Clone the repository
git clone https://github.com/aryxn-builds/portfolio.git

# Navigate to project
cd portfolio

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
```

### Environment Variables

```env
# Required for AI Chat Assistant
ANTHROPIC_API_KEY=your_anthropic_api_key_here

# Optional вҖ” increases GitHub API from 60 to 5000 req/hour
GITHUB_TOKEN=your_github_token_here
```

### Development

```bash
npm run dev
# Opens at http://localhost:3000
```

### Production Build

```bash
npm run build
npm run start
```

### Deploy to Vercel

```bash
npx vercel --prod
```
